# SmartLanguage

## Présentation

SmartLanguage est une application orientée apprentissage linguistique conçue pour proposer une expérience personnalisée
aux utilisateurs afin de leur permettre d'évaluer et d'améliorer leurs compétences en langues étrangères.

L'idée et d'avoir une ia qui pose une question vocale afin qu'on puisse y répondre à l'oral et que l'ia puisse évaluer
notre réponse et nous donner une note.

L’objectif principal est de proposer un outil simple, rapide et scalable, déployé dans une architecture cloud moderne.

---

## Fonctionnalités

La plateforme permet de :

- Générer et gérer du contenu pédagogique dynamique
- Adapter les exercices selon le niveau utilisateur
- Sauvegarder la progression et l’historique
- Authentifier les utilisateurs de manière sécurisée
- Exposer une API backend pour consommation frontend

## Stack technique

| TypeScript   | Typage statique          |
| Tailwind CSS | UI moderne et responsive |

| Vite         | Build tool rapide        |

| Technologie          | Usage                 |
|----------------------|-----------------------|
